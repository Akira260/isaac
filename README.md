# 📋 以撒的结合：忏悔 · 全成就道具解锁清单

一款专为《以撒的结合：忏悔》玩家打造的**便捷性轻量化进度管理工具**。基于 B 站 UP 主的道具推荐解锁攻略制作，旨在帮助你从混乱的成就中解脱，快速规划最强道具解锁路线！

✨ **[ 点击立即在线使用 ](https://akira260.github.io/isaac/)**

---

## 🚀 核心功能

* **🎯 动态进度追踪**：百分比显示全局进度条，完成进度一目了然。
* **🔥 优先级视觉高亮**：精选“红色核心级”推荐道具并附带「优先」标签，支持一键过滤，直奔关键目标。
* **📱 完美响应式适配**：深色暗黑级 UI 设计，无论是电脑端爆肝还是手机端盲刷，都能完美适配、舒适护眼。
* **💾 本地免密保存**：基于 `localStorage` 技术，勾选状态自动实时保存在本地浏览器中，关闭或重开网页进度绝不丢失。
* **🖱️ 极致交互体验**：告别繁琐的勾选框！点击成就整行即可直接触发勾选，操作丝滑顺畅。
* **🧹 一键重置清空**：底部贴心设置“清空所有进度”按钮，方便开启新档或重新规划。

## ⚠️ 注意事项

> 💡 **关于数据保存**：由于本工具数据存储在浏览器的本地缓存（LocalStorage）中，**清除浏览器缓存或使用隐私无痕模式**会导致勾选的进度丢失，请谨慎清理。
